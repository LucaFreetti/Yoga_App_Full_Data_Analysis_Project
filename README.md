
# 🧘 Yoga App — Data Analysis Project

> A comprehensive data analysis aimed at improving the app across all dimensions: user growth, engagement, perceived quality, and coach performance.

---

## 📋 Project Overview

This project is the **final exam assignment** for a Data Analysis course. It analyzes data from a yoga app operating across Italy, the UK, and France (2020–2023). The goal is to identify strategic levers to scale the platform, improve user retention, and raise the overall quality of the experience — for both users and coaches.

The analysis covers four main areas:
- **Growth** — lessons, active users, and subscriptions over time
- **Perceived Quality** — ratings trends, seasonal patterns, and scalability risks
- **Coaches** — performance gaps and language-specific issues
- **Users** — levels, lesson categories, platforms, and countries

## 🔬 Methodology

The analysis followed a structured, end-to-end data analysis workflow:

### 1. 📥 Data Loading
The raw dataset (`Lifestyle_data_Final_Project.csv`) was loaded into a Pandas DataFrame. Initial inspection covered shape, data types, missing values, and basic distributions.

### 2. 🧹 Data Cleaning & Transformation
- Handled missing values (notably: subscription data for 2023 was absent)
- Standardized column formats (dates, categorical labels, language codes)
- Removed duplicates and validated consistency across records
- Exported the cleaned version as `Lifestyle_data_Final_Project_modify.csv` for use in Tableau

### 3. 🔍 Exploratory Data Analysis (EDA)
- Computed growth metrics: YoY lessons, active users, subscriptions per quarter
- Analyzed rating distributions by coach, language, category, and time period
- Identified seasonal patterns in user ratings (Q3 vs Q4)
- Segmented users by country, platform (TV/PC/Mobile), level, and lesson language

### 4. 📈 Visualization
- **Matplotlib** and **Seaborn** were used to produce analytical charts directly in the notebook
- **Tableau** was used to build an interactive, stakeholder-facing dashboard from the cleaned data

### 5. 💡 Insight Generation
Findings were synthesized into 7 strategic recommendations, each grounded in the data and prioritized by business impact.

## 🔍 Key Findings

### 📈 Growth
- Active users grew **×3.8** from 2021 to 2023 (55 → 209 users)
- Lessons grew **×6.2** in the same period (90 → 559 lessons)
- Subscriptions fluctuate between 15–26 per quarter with no clear trend — a sign of increasing **stickiness**: existing users are engaging more over time
- *Note: subscription data for 2023 is missing*

### ⭐ Perceived Quality
- The global average rating is stuck at **3.02/5** across all quarters — no improvement trend
- Seasonal pattern: Q4 performs best (~3.2), Q3 worst (~2.8)
- At the current growth rate, a stagnant 3/5 rating poses a serious **reputational risk**

### 🏆 Coaches
- Top-rated coach: **~3.6/5** — bottom-rated: **~2.58/5**
- Coach 106 shows a specific issue with French-language lessons (all ratings below 2.7 stars)
- No coach exceeds 3.6/5 → the quality problem is **systemic**, not isolated to a few individuals

### 🧘 Users
- **47% of lessons are watched on TV** — the large-screen experience is the top UX priority
- Advanced-level lessons dominate (437 vs 236 intermediate) — expert users show the highest retention
- Italy drives the most engagement: 465 out of 999 lessons conducted in Italian
- 178 users in 2023 took lessons in more than one language — suggesting **unmet multilingual demand**

## 💡 Strategic Recommendations

| # | Lever | Action |
|---|-------|--------|
| 01 | **TV-First Strategy** | Prioritize TV app UX — 47% of lessons happen on the big screen |
| 02 | **Power Yoga + Advanced** | Expand this combo with top coaches — it's the core user segment |
| 03 | **Boost Ratings** | Review coach training and lesson formats to break the 3/5 ceiling |
| 04 | **Meditation** | Only 148 lessons in a high-demand wellness category — enormous untapped potential |
| 05 | **Re-engagement** | Introduce streaks, notifications, and programs to bring back one-lesson users |
| 06 | **Onboarding Form** | Match users to the best-fit coach at sign-up |
| 07 | **Review Analysis** | Investigate root causes behind 1 and 2-star ratings |
